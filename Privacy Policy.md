Privacy Policy

Last updated: July 10, 2026

This Privacy Policy explains how this Discord bot ("the Bot") handles user data.

Data We Collect

The Bot does not collect personal data, message history, or user activity logs.

The Bot may store only data required for its functionality, such as:

Discord server configuration settings.
User preferences and feature settings.
API keys or tokens manually provided by users for third-party services.
Other data explicitly configured by users through Bot commands or settings.
How We Use Data

Stored data is used only to:

Provide Bot features.
Remember user and server configurations.
Authenticate with third-party services requested by the user.

We do not use stored data for advertising, tracking, profiling, or selling purposes.

Data Sharing

We do not sell or share stored data with third parties.

API keys and user-provided credentials are only used for the functionality they were provided for.

Data Storage and Security

Data is stored securely and access is limited to the Bot's operation. However, no method of storage can guarantee complete security.

Users should not provide API keys or credentials they do not want the Bot to access.

Data Removal

Users can request removal of stored data by contacting the Bot developer. Requests will be handled manually where possible.

Changes

This Privacy Policy may be updated when the Bot's functionality changes.
